# 🎵 Spotify App Clone

A simple Spotify-like music player built using HTML, CSS, and JavaScript.

# 🚀 Features
- Play and pause music
- Music player UI similar to Spotify
- Interactive controls
- Clean and responsive design

# 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

# ⚙️ How It Works
- Click on a song
- Press the play button
- Enjoy your music

# 📌 Note

This is a front-end project created for learning purposes and does not include real streaming services.
